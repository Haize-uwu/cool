1. Propositional Logic

\(A \implies B\) This evalutes true if A is false OR B is true

1.1. Logical Constants

\(\top\): True propostion

\(\bot\): False propstion

1.2. Variables

  • p, q, r : lower case roman letters.

1.3. Construction Trees

  • Taking any subtree is a valid logical formulae itself
  • the size of the graph is the number of nodes (variables + connectives)
  • The main connective is the one applied last, non-leaf node
  • create the tree from the inside

1.4. Parsing Trees

  • Top to bottom

1.6. Tautologies

  • propositionnaly valid formula that is always True.

1.6.1. Notation

  • \(\models A\)
  1. Examples
    • \(\models\) p ∨ ¬ p - Law of excluded middle
    • \(\models\) ¬(p ∧ ¬ p) - Law of non-contradiction
    • \(\models\) ((p ∧(p \(\implies\) q))\(\implies\) q) - modus ponents
      • Set q false, left must be true so all true.

1.7. Contradictions

  • always false formula
  1. If a formula is not a contradiction, then it is satisfiable

1.8. Propositional Logical Consequence

\[A_1 , A_2 ,..., A_n \models C\]

  • if C is true whenever all \(A_i\) are true.
  • Tautology is a special case of consequence

\[ \phi \models C = \models C\]

1.8.1. Consiquence is reducible to validity

\[A_1 , ... , A_n \models B \] is equivalent to: \(A_1 \land ... \land A_n \models B\)
then \[\models (A_1 \land ... \land A_n) \implies B\]

1.9. Sound Rules of propositional inference

  • A rule of propositional inference is a scheme where \(P_i\) are the premises we assume and \(C\) is the conclusion \[\frac{P_1 , ..., P_n}{C}\]
  • An inference rule is logically sound if \[P_1 , ..., P_n \models C\]
  • Putting concrete propostions (substituting for variables) is a propositional inference.

1.10. How to check correctness

Check if a rule is sound : \[\frac{S, S \implies h}{h}\] so check if \(S, S \implies h \overset{?}{\models} h\)
∴ the inference is correct based on a sound rule, but can’t say rules is sound

1.11. Fallacies of Implication (Derivative Implication)

  • converse: \(B \implies A\)
  • inverse: ¬ A \(\implies\) ¬ B
  • contrapositve: ¬ B \(\implies\) ¬ A this is sound.

1.12. Truth assignments in propostional logical

  • PVAR : propostional variables (countably infinite)
  • FOR : set of all propositional formulas

A truth assignment is a map \\ v : PVAR → {T, F} and
\(\bar{v}\):FORM →{T,F}

  1. Note the following:
    • \(\bar{v}(\top)\) = T
    • \(\bar{v}(\bot)\) = F
    • \(\bar{v}(p)\) = v(p)

1.13. Logical Equivalence

A ≡ B both obtain the same truth value under all truth valuations
\[A \equiv B \iff \bar{v}(A) = \bar{v}(B)\] for all truth assignments if v

1.13.1. De’ Morgans Law

¬(p∧ q) ≡ ¬ p ∨ ¬ q

1.13.2. this one

p ∧ (p ∨ q) ≡ p ∧ p ≡ p\

1.14. Relating Logical Equivalence, consequence and validity

  1. Reducible to validity

    \[A \equiv B\ iff \models A \iff B \]

    1. Proof in notes

      Suppose A ≡ B
      Let v:PVAR→ {T,F} be arbitrary
      since A ≡ B, \(\bar{v}\)(A) = \(\bar{v}\)(B)
      ∴ \(\bar{v}\)(A \(\iff\) B) = T
      Since v was arbitrary we conclude
      \(\models\) A \(\iff\) B

  2. Reducible to Logical Consequence

    A ≡ B iff both A \(\models\) B and \(B \models A\)

1.15. ≡ is an equivalence relation

  1. reflexive A≡ A
  2. symmetric if A≡ B then B ≡ A
  3. transitive if A ≡ B and B ≡ C then A ≡ C

1.16. ≡ is a congruence wrt to logical connectives

  • if A ≡ B then ¬ A ≡ ¬ B
  • if \(A_1 \equiv B_1\) and \(A_2 \equiv B_2\) then (A1 ⋅ A2 ) ≡ (B1 ⋅ B2) for all ⋅ = {∨, ∧, \(\implies\), \(\iff\)}
  1. Theorem Equivalent replacement

    if A ≡ B then C(A/p) ≡ C(B/p)
    We replace p with A all at once, the same for B

    1. Proof in notes → by induction on C

      By Induction on C:

      1. \(C\) is \(\bot\) Then \(C(A/p)\) = \(\bot\) and C(B/p) = \(\bot\)

      ∴ \(C(A/p) = \bot \equiv \bot = C(B/p)\)

      1. \(C\) is \(\top\) same as above.
      2. 1. \(C\) is \(q\) where \(q \neq p\), then \(C(A/p) = q\) and \(C(B/p) = q\)

      SO \(C(A/p) = q \equiv q = C(B/p)\)

      1. 2. \(C\) is \(p\) , then

      \(C(A/p) = A \equiv B = C(B/p)\)
      Inductive Hypothesis: Suppose holds for formulas \(C_1\) and \(C_2\)
      \(C_1 (A/p) \equiv C_1 (B/p)\) and
      \(C_2(A/p) \equiv C_2(B/p)\)

      1. C is ¬ C :

      Then \(C(A/p) = \not C(A/p) \overset{\star}{\equiv} \not C(B/p) =C(B/p)\),
      Where * holds by the Inductive Hypothesis and the fact that ≡ is a congruence wrt to propositional connectives.

      1. \(C\) is \((C_1 \cdot C_2)\) for some \(\cdot = \{\land \lor \implies \iff \}\)

      Then \(C(A/p) = (C_1(A/p) \cdot C_2(A/p))\) but by inductive hypothesis
      \(C_1 (A/p) \equiv C_1 (B/p)\) and
      \(C_2(A/p) \equiv C_2(B/p)\)
      So by the fact that ≡ us a congruence wrt to propositional connectives;
      \(C(A/p) = ((C_1(A/p) \cdot C_2(A/p))\equiv(C(A/p) = (C_1(B/p) \cdot C_2(B/p))=C(B/p)\)
      ∴ C(A/p) ≡ C(B/p)
